# Active Directory Lab

## Overview
A full IT support project demonstrating deployment and administration of a Windows Active Directory environment inside a virtual lab.

## Skills Demonstrated
- Domain Controller deployment
- Active Directory Users & Computers (ADUC)
- Organizational Unit (OU) structure planning
- Group Policy configuration (GPOs)
- File share permissions
- User onboarding workflow
- PowerShell automation for AD tasks

## Components
- DC setup logs
- Screenshots of ADUC & GPO
- PowerShell automation scripts
- Documentation and topology diagrams
